# Anon-Net

Anon-Net是一個基於Tor的匿名網路工具，利用polipo實現Tor網路代理。



### 截圖：

![](https://s1.ax1x.com/2020/10/23/BAdrGT.png)



### 安裝：

您必須首先安裝polito;

```bash
git clone https://github.com/ixiniansec/Anon-Net.git
cd Anon-Net
python3 anonnet.py
```



### 關於IP切換：

修改`MaxCircuitDirtiness`的值，但是請注意，這不一定能夠獲得新的IP地址。
